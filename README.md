# Connect 4 AI using Minimax with Alpha-Beta Pruning

## Overview
This project implements an AI-powered Connect 4 game where the player competes against an intelligent agent using the Minimax algorithm with Alpha-Beta Pruning.

## Objective
To build an optimal decision-making AI capable of evaluating game states and selecting the best possible move.

## Problem Statement
The AI must analyze possible future moves and choose the optimal strategy to maximize its chances of winning while minimizing the opponent's advantage.

## Features
- Human vs AI gameplay
- Board state evaluation
- Minimax decision-making
- Alpha-Beta pruning optimization
- Winning state detection

## Algorithms Used

### Minimax Algorithm
Used for recursive game tree exploration.

### Alpha-Beta Pruning
Optimizes Minimax by eliminating unnecessary branches.

## Implementation Details
- Developed in Python
- Executed in Google Colab
- NumPy used for board representation
- Dynamic game state analysis
